# Astra Linux: Особенности Форензики

### Команда
```bash
astra-mic-control status
```
Что делает: проверяет состояние механизма MIC.
Параметры: `status` - текущий статус.

### Команда
```bash
pdpl-user <user>
```
Что делает: показывает уровни/категории пользователя.
Параметры: `<user>` имя пользователя.

### Команда
```bash
pdpl-file /var/log
```
Что делает: показывает метки объекта ФС.
Параметры: путь к объекту.

### Команда
```bash
usercaps -M <user>
```
Что делает: показывает мандатные/спец-возможности пользователя.
Параметры: `-M` расширенный режим вывода.
