# Linux CTF Чеклист

### Команда
```bash
date -u
```
Что делает: фиксируем время старта расследования.
Параметры: `-u` UTC.

### Команда
```bash
ps aux --sort=-%cpu | head -n 30
```
Что делает: быстрый обзор процессов.
Параметры: `--sort=-%cpu` сортировка по CPU.

### Команда
```bash
ss -tulpen
```
Что делает: открытые порты/процессы.
Параметры: `-t/-u/-l/-p/-e/-n`.

### Команда
```bash
grep -RinE 'flag\{|thm\{|ctf\{' /home /root /opt /var/www 2>/dev/null
```
Что делает: быстрый поиск флагов по типовым путям.
Параметры: `-R` рекурсивно, `-n` строки, `-E` regex.
