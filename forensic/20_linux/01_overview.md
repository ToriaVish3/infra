# Linux Форензика: Общий Порядок

## 1) Время и контекст

### Команда
```bash
date -u
```
Что делает: фиксирует UTC-время расследования.
Параметры: `-u` - UTC.

### Команда
```bash
cat /etc/os-release
```
Что делает: определяет дистрибутив и версию системы.
Параметры: нет.

## 2) Пользователи и входы

### Команда
```bash
who
```
Что делает: текущие сессии пользователей.
Параметры: нет.

### Команда
```bash
last -a | head -n 30
```
Что делает: история успешных входов.
Параметры: `-a` - IP/host в конце.

## 3) Процессы и сеть

### Команда
```bash
ps aux --sort=-%mem | head -n 30
```
Что делает: подозрительные процессы по памяти.
Параметры: `--sort=-%mem` - сортировка по памяти.

### Команда
```bash
ss -tulpen
```
Что делает: открытые порты и соединения.
Параметры: `-t/-u/-l/-p/-e/-n`.

## 4) Persistence

### Команда
```bash
crontab -l
```
Что делает: cron текущего пользователя.
Параметры: `-l`.

### Команда
```bash
systemctl list-unit-files --type=service | grep enabled
```
Что делает: сервисы автозапуска.
Параметры: `--type=service`.
